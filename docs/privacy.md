# WindroseBot Privacy Policy

**Last Updated:** April 18, 2026

## What Data We Collect

WindroseBot does **not** collect, store, or share any personal user data.

## What the Bot Accesses

The Bot accesses the following information solely to provide its functionality:

- **Discord interactions** — The Bot receives slash command inputs (command name and parameters) to respond to your requests. These are processed in real time and not stored.
- **Nitrado API data** — The Bot queries the Nitrado API for server status, backup lists, and log files. This data is used only to display information in Discord and is not stored permanently.
- **Server status** — The Bot temporarily holds the current server status in memory to detect status changes. This is cleared when the Bot restarts.

## What We Do NOT Collect

- User messages or message content
- Usernames, user IDs, or any personal identifiers
- Server member lists
- Voice or media data
- Analytics or tracking data

## Third-Party Services

The Bot connects to the following third-party services:

- **Discord API** — To receive commands and send messages. Subject to [Discord's Privacy Policy](https://discord.com/privacy).
- **Nitrado API** — To monitor and manage the gameserver. Subject to [Nitrado's Privacy Policy](https://nitra.do/Privacy).

## Data Storage

No user data is stored on disk, in databases, or in any external service. The Bot runs entirely in memory.

## Children's Privacy

The Bot does not knowingly collect any data from users under the age of 13.

## Changes

This privacy policy may be updated at any time. Continued use of the Bot constitutes acceptance of any changes.

## Contact

For questions or concerns about this privacy policy, reach out to the server administrator.
